# Online Book Store – E-Commerce Website

This is a static e-commerce website created as part of the Major Project.
It simulates an online book store where users can view books, search,
add items to cart and place an order.

## Features
- Home page with banner and featured books
- Products page with all books displayed in grid
- Product details page with complete book information
- Add to cart and cart management
- Total price calculation
- Order and order success pages
- Dark mode toggle
- Product search bar
- CSS animations

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Folder Structure
project/

## How to Run
Open index.html in any web browser.
